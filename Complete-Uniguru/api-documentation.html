📖 Overview

UniGuru is an AI-powered chatbot platform with RAG (Retrieval-Augmented Generation) capabilities and integrated voice features through the Vaani LM service.

Base URLs:
• UniGuru Backend: https://complete-uniguru.onrender.com
• Vaani LM Service: https://vaani-sentinel-gs6x.onrender.com

POST Register User
https://complete-uniguru.onrender.com/api/v1/auth/register

Request Body:

POST Login User
https://complete-uniguru.onrender.com/api/v1/auth/login

Request Body:

{ "email": "john@example.com", "password": "password123" }

💬 Chat System

POST Send Chat Message
https://complete-uniguru.onrender.com/api/v1/chat/new

Send a message to a guru and get AI response

Request Body:

{ "message": "Explain quantum physics", "chatbotId": "GURU_ID_HERE", "userId": "YOUR_USER_ID", "generateAudio": true }
GET Get User Chats
https://complete-uniguru.onrender.com/api/v1/chat/list

Get all chats for the authenticated user

POST Create New Chat
https://complete-uniguru.onrender.com/api/v1/chat/create

Create a new chat session with a guru

Request Body:

{ "guruId": "GURU_ID_HERE", "title": "My Physics Discussion" }
POST Generate Audio for Chat Message
https://complete-uniguru.onrender.com/api/v1/chat/audio

Generate audio response for a chat message using Vaani LM service

Request Body:

{ "message": "Explain quantum physics in simple terms", "chatbotId": "GURU_ID_HERE", "userId": "YOUR_USER_ID", "voice": "en_us_female_conversational", "language": "en" }

Response:

{ "success": true, "audioUrl": "https://vaani-sentinel-gs6x.onrender.com/audio/generated_audio.mp3", "text": "Quantum physics is the study of matter and energy at the smallest scales...", "metadata": { "voice": "en_us_female_conversational", "language": "en", "duration": 15.2 } }

🎵 Vaani LM Service

The Vaani service provides advanced AI capabilities including text-to-speech generation.

Note: Vaani service endpoints require separate authentication. Use the login endpoint first to get a token.

🔐 Vaani Authentication

POST Vaani Login
https://vaani-sentinel-gs6x.onrender.com/api/v1/auth/login

Authenticate with Vaani service

Request Body:

{ "username": "admin", "password": "secret" }

🤖 AI Content Generation

POST Generate TTS Audio
https://vaani-sentinel-gs6x.onrender.com/api/v1/agents/tts

Generate text-to-speech audio with intelligent caching

Request Body:

{ "text": "Hello, this is a test message for TTS generation.", "voice": "en_us_female_conversational", "language": "en" }
POST Generate Platform Content
https://vaani-sentinel-gs6x.onrender.com/api/v1/agents/generate-content

Generate platform-specific content from existing content

Request Body:

{ "content_id": "CONTENT_ID_HERE", "platforms": ["twitter", "instagram", "linkedin"], "tone": "uplifting", "language": "en" }
POST Generate Voice Content
https://vaani-sentinel-gs6x.onrender.com/api/v1/agents/generate-voice

Generate voice scripts for audio content

Request Body:

{ "content_id": "CONTENT_ID_HERE", "language": "hi", "tone": "devotional", "voice_tag": "hi_in_female_devotional" }

🧪 Testing with Postman

Step-by-Step Guide

  1. Import to Postman: Copy any cURL command and use "Import" → "Raw text" in Postman
  2. Get UniGuru Token: First run the login endpoint to get your JWT token
  3. Get Vaani Token: Login to Vaani service separately to get its JWT token
  4. Set Authorization: Add "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiJhZG1pbiIsImV4cCI6MTc2NDc1MTI5NH0.AyPO0qrrSaApD1HfxK10MYygqZgzijMzrllxGUxMxVw" to headers
  5. Test Endpoints: Start with health checks, then authentication, then other endpoints
Quick Start Example:
1. Login to UniGuru: POST /api/v1/auth/login
2. Login to Vaani: POST https://vaani-sentinel-gs6x.onrender.com/api/v1/auth/login
3. Create a guru: POST /api/v1/guru/custom-guru/YOUR_USER_ID
4. Send a chat message: POST /api/v1/chat/new
5. Generate TTS audio: POST https://vaani-sentinel-gs6x.onrender.com/api/v1/agents/tts

Common Response Codes

Code Description
200 Success - Request completed successfully
400 Bad Request - Invalid input data
401 Unauthorized - Invalid or missing JWT token
403 Forbidden - Access denied
404 Not Found - Resource doesn't exist
500 Internal Server Error - Server-side issue
